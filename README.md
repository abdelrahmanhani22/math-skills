# Math Skills

## Description
`math-skills` is a Go program that reads a text file containing a list of integers and calculates the following statistical values:

- Average
- Median
- Variance
- Standard Deviation

Each value is rounded to the nearest integer, and the output format matches the reference program provided in the project instructions.

---

## Input Format
- The program takes **one argument**: the path to a text file.
- The file must contain **integers separated by spaces and/or new lines**.
